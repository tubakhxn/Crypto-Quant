# Crypto Market Making Engine

## Dev/Creator

tubakhxn

## Overview

Professional crypto market making simulator with inventory-aware spread generation, synthetic order book simulation, volatility-based quoting, and cinematic dashboards.

## Features

- Cinematic dark institutional dashboard
- Synthetic fallback market data
- Modular quant architecture
- Research-oriented analytics
- Interactive visualizations
- Real-time simulation workflows
- Production-style folder structure

## Folder Structure

```text
Crypto-Market-Making-Engine/
├── app.py
├── main.py
├── requirements.txt
├── README.md
├── core/
├── data/
└── visuals/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

or

```bash
python main.py
```

## Technologies Used

Python, NumPy, Pandas, Plotly, Streamlit
