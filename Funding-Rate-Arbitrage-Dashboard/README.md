# Funding Rate Arbitrage Dashboard

## Dev/Creator

tubakhxn

## Overview

Cross-exchange funding rate analytics platform with opportunity scanning, synthetic exchange feeds, and profitability estimation.

## Features

- Cinematic dark institutional dashboard
- Synthetic fallback market data
- Modular quant architecture
- Research-oriented analytics
- Interactive visualizations
- Real-time simulation workflows
- Production-style folder structure

## Folder Structure

```text
Funding-Rate-Arbitrage-Dashboard/
├── app.py
├── main.py
├── requirements.txt
├── README.md
├── core/
├── data/
└── visuals/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

or

```bash
python main.py
```

## Technologies Used

Python, CCXT, Pandas, Plotly, Streamlit
