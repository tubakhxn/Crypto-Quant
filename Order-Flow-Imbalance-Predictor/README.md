# Order Flow Imbalance Predictor

## Dev/Creator

tubakhxn

## Overview

Research-oriented order flow imbalance predictor using simulated liquidity pressure and machine learning forecasting.

## Features

- Cinematic dark institutional dashboard
- Synthetic fallback market data
- Modular quant architecture
- Research-oriented analytics
- Interactive visualizations
- Real-time simulation workflows
- Production-style folder structure

## Folder Structure

```text
Order-Flow-Imbalance-Predictor/
├── app.py
├── main.py
├── requirements.txt
├── README.md
├── core/
├── data/
└── visuals/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

or

```bash
python main.py
```

## Technologies Used

Python, NumPy, Scikit-learn, Plotly, Streamlit
