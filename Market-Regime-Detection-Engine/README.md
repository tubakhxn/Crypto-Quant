# Market Regime Detection Engine

## Dev/Creator

tubakhxn

## Overview

Quantitative market regime classification engine using clustering, volatility metrics, and trend analytics.

## Features

- Cinematic dark institutional dashboard
- Synthetic fallback market data
- Modular quant architecture
- Research-oriented analytics
- Interactive visualizations
- Real-time simulation workflows
- Production-style folder structure

## Folder Structure

```text
Market-Regime-Detection-Engine/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

or

```bash
python main.py
```
