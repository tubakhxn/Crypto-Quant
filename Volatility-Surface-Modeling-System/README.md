# Volatility Surface Modeling System

## Dev/Creator

tubakhxn

## Overview

Interactive volatility surface modeling engine with synthetic options chain generation and 3D visual analytics.

## Features

- Cinematic dark institutional dashboard
- Synthetic fallback market data
- Modular quant architecture
- Research-oriented analytics
- Interactive visualizations
- Real-time simulation workflows
- Production-style folder structure

## Folder Structure

```text
Volatility-Surface-Modeling-System/
├── app.py
├── main.py
├── requirements.txt
├── README.md
├── core/
├── data/
└── visuals/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

or

```bash
python main.py
```

## Technologies Used

Python, NumPy, SciPy, Plotly, Pandas
